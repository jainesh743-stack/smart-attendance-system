# Smart Attendance System
BCA Final Year Project — Flask + QR + OTP + Face Recognition

## Deploy on Railway
1. Push this repo to GitHub
2. Go to railway.app → New Project → Deploy from GitHub
3. Done! ✅

## Run Locally
```
pip install -r requirements.txt
python app.py
```

## Admin Login
- Username: admin
- Password: admin123

## How it works
1. Admin starts session → QR + OTP generated
2. Student logs in → Scans QR → Enters OTP → Face scan
3. Attendance marked ✅
